# Como buildar

**É necessário: PNPM.**<br><br>
Rode o comando `pnpm install && pnpm build`, em seguida o **executável estará na pasta raiz do projeto**.<br>
Após isso, você pode mover o executável para algum lugar de preferência e apagar a pasta do projeto.

<br>Utiliza [AppImageTool](https://github.com/AppImage/appimagetool).
